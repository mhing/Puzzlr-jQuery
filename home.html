Goal

The goal is to move all the puzzle pieces into their correct position, thus revealing a completed image.

Moving

Click a piece neighboring the empty slot (black square) to slide the piece into the empty slot.

Click the Shuffle Remaining Pieces button to shuffle the pieces and continue to solve the puzzle. Can only be used once per game.

Click the Reset Button to restart the game.

Winning

When you win the game, your score will be displayed and you will have the option to select a new image or restart the game with the default image.

Software

The frontend relies on the use of the jQuery and jQUery UI libraries and the Bootstrap toolkit.


Game Play

For now, Puzzlr only supports playing with a 3x3 puzzle using the default image.

The next two iterations will add the ability to use user selected images and then the ability to solve a 4x4 and 5x5 puzzle.
